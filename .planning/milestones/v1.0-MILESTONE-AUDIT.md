## Gap Closure Summary (from Prior Audit)

### Gap 1: CORE-04 — Provenance __getattr__ incomplete (CRITICAL)

**Prior finding:** `provenance/__init__.py.__getattr__` only delegated to `provenance.py`. `AnalysisStep` in `analysis_ir.py` was unreachable via `from lobster.core.provenance import AnalysisStep`, breaking 6 runtime call sites in `de_analysis_expert.py`.

**Resolution:** Phase 10 extended `__getattr__` to search all 4 submodules via `_SUBMODULES` tuple (`provenance.py`, `analysis_ir.py`, `lineage.py`, `ir_coverage.py`). Verified: `AnalysisStep`, `ParameterSpec`, `LineageMetadata`, `IRCoverageAnalyzer`, `ProvenanceTracker` all resolve correctly. 44 shim tests pass. **Status: CLOSED.**

### Gap 2: DMGR-04 — CI guard ineffective (MEDIUM)

**Prior finding:** `.github/workflows/ci-basic.yml` grep step used `|| true`, making the deprecated-import guard informational only. 39 pre-existing violations in `packages/` were never blocked.

**Resolution:** Phase 11 replaced `|| true` with strict `if grep ... then exit 1; fi` pattern. All 39 deprecated imports in `packages/` migrated to canonical `lobster.core.runtime.data_manager` path. Structural pytest added for local enforcement. Zero deprecated imports remain. **Status: CLOSED.**
