<objective>
Harden tar extraction security with reject-all policy and fix download orchestrator race condition with compare-and-swap status transitions.

Purpose: Current archive extraction silently skips unsafe members (allowing partial malicious extraction) and lacks symlink escape detection (CVE-2007-4559). The download orchestrator sets IN_PROGRESS without checking current status, so two concurrent workers can both claim the same entry. These are the two security/correctness bugs that affect data integrity and system safety.

Output: Reject-all archive security with symlink checks and logged offending paths; CAS-based status transitions preventing duplicate worker processing.
</objective>

       ```python
       def _is_safe_member(self, member: tarfile.TarInfo, target_dir: Path) -> bool:
           """Prevent path traversal and symlink escape attacks during TAR extraction."""
           # 1. Check symlinks and hardlinks
           if member.issym() or member.islnk():
               link_target = Path(member.linkname)
               if link_target.is_absolute():
                   logger.warning(f"Blocked absolute symlink: {member.name} -> {member.linkname}")
                   return False
               # Resolve relative link against member's parent directory
               member_parent = (target_dir / Path(member.name)).parent
               resolved_link = (member_parent / link_target).resolve()
               if not str(resolved_link).startswith(str(target_dir.resolve())):
                   logger.warning(f"Blocked symlink escaping target: {member.name} -> {member.linkname}")
                   return False

           # 2. Check path traversal (existing logic, preserved)
           member_path = Path(member.name)
           try:
               target_path = (target_dir / member_path).resolve()
               common_path = Path(os.path.commonpath([target_dir.resolve(), target_path]))
               is_safe = common_path == target_dir.resolve()
               if not is_safe:
                   logger.warning(f"Blocked unsafe TAR member: {member.name}")
               return is_safe
           except (ValueError, RuntimeError) as e:
               logger.warning(f"Path safety check failed for {member.name}: {e}")
               return False
       ```

    2. In `lobster/core/download_queue.py`, update `update_status` (line 144):

       Add `expected_current_status` parameter. Return `Optional[DownloadQueueEntry]` instead of `DownloadQueueEntry` (where None = CAS failed). Per Pitfall 3 from RESEARCH.md, this is backward compatible because callers that don't pass expected_current_status get the old behavior, and callers that do check the return can distinguish None (failed) from entry (succeeded).

       ```python
       def update_status(
           self,
           entry_id: str,
           status: DownloadStatus,
           expected_current_status: Optional[DownloadStatus] = None,
           modality_name: Optional[str] = None,
           error: Optional[str] = None,
           downloaded_by: Optional[str] = None,
       ) -> Optional[DownloadQueueEntry]:
           """Update entry status. Returns entry on success, None if CAS precondition failed."""
           with self._locked():
               entries = self._load_entries()
               for entry in entries:
                   if entry.entry_id == entry_id:
                       # CAS check
                       if expected_current_status is not None:
                           current = entry.status
                           # Handle both enum and string comparison
                           current_val = current.value if hasattr(current, 'value') else current
                           expected_val = expected_current_status.value if hasattr(expected_current_status, 'value') else expected_current_status
                           if current_val != expected_val:
                               logger.info(
                                   f"CAS failed for entry '{entry_id}': "
                                   f"expected {expected_val}, actual {current_val}"
                               )
                               return None
                       entry.update_status(
                           status=status, modality_name=modality_name,
                           error=error, downloaded_by=downloaded_by,
                       )
                       self._backup_queue()
                       self._write_entries_atomic(entries)
                       logger.debug(f"Updated entry {entry_id} status to {status}")
                       return entry
               raise EntryNotFoundError(f"Entry '{entry_id}' not found in queue")
       ```
