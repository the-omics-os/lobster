Wire AquadifMonitor into the existing callback chain: client.py constructs it, graph.py builds the tool metadata map and sets it on DataManagerV2, TokenTrackingCallback calls record_tool_invocation, DataManagerV2 calls record_provenance_call.

Purpose: Connect the tested AquadifMonitor service (Plan 01) to the runtime system so that tool category usage and provenance compliance are tracked during actual sessions. The wiring follows a single-injection-point design to prevent double-counting in cloud sessions.

Output: Modified graph.py, callbacks.py, data_manager_v2.py, client.py with monitor integration
</objective>

    After the graph metadata section (after line ~675, before the final return), add a new section:

    ```python
    # ==========================================================================
    # AQUADIF Monitoring: build tool_metadata_map and wire monitor
    # ==========================================================================
    if aquadif_monitor is not None:
        tool_metadata_map: Dict[str, Dict[str, Any]] = {}

        # Collect from supervisor-accessible handoff tools (DELEGATE metadata)
        for t in agent_tools:
            if hasattr(t, "metadata"):
                tool_metadata_map[t.name] = t.metadata

        # Collect from shared workspace tools
        for t in [list_available_modalities, get_content_from_workspace,
                  delete_from_workspace, execute_custom_code]:
            if hasattr(t, "metadata"):
                tool_metadata_map[t.name] = t.metadata

        # Collect from all child agent tools (PregelNode traversal)
        for agent_name, agent in created_agents.items():
            try:
                tools_node = agent.nodes.get("tools")
                if tools_node and hasattr(tools_node, "runnable"):
                    for t in (tools_node.runnable.tools or []):
                        if hasattr(t, "metadata"):
                            tool_metadata_map[t.name] = t.metadata
            except Exception:
                pass  # Fail-open: skip agents whose tools can't be extracted

        # Populate monitor's tool map (monitor already attached to callback by reference)
        aquadif_monitor._tool_metadata_map = tool_metadata_map

        # Wire monitor to DataManagerV2 (setattr avoids changing DM constructor)
        data_manager._aquadif_monitor = aquadif_monitor

        logger.debug(
            f"AQUADIF monitor: {len(tool_metadata_map)} tools mapped"
        )
    ```

    Also add `Dict` to the typing imports at the top of graph.py if not already present (it likely is).
